## 準備
`pip install pyxel==1.5.8 numpy`  

## 実行
`python main.py`
or
`pyxel play app/mini_shooting.pyxapp`

## 操作方法
移動(move):矢印キー(arrow key)  
低速移動(slow):左Shift(left shift)  
ショット(shot):Z  
中断(quit mission):Q
